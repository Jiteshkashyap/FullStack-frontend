# Auth Project – Full Stack Application

This is a full-stack authentication project with a React frontend and a Node.js + Express backend. The project demonstrates a basic authentication flow with proper separation of frontend and backend.

---

## Project Structure

auth/
├── frontend/
└── backend/

---

## Tech Stack

Frontend:
- React (Vite)
- Axios

Backend:
- Node.js
- Express
- MongoDB
- JWT Authentication

---

## Backend Configuration

- Backend runs on port **5000**
- Base URL:
  http://localhost:5000

Environment variables used in backend:
- PORT=5000
- MONGO_URI
- JWT_SECRET



## Frontend Configuration

- Frontend runs on port **5173**
- URL:
  http://localhost:5173

Frontend API requests are configured to communicate with the backend at:
http://localhost:5000

---

## How to Run the Project

1. Start the backend server (port 5000)
2. Start the frontend server (port 5173)
3. Open browser and visit:
   http://localhost:5173
